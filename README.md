# 🌿 Nature's Platter  

**Nature's Platter** is a beautifully designed responsive website built using **Tailwind CSS** and **DaisyUI**. This project is created as part of an exercise and assignment to practice modern frontend development techniques.  

## 🛠️ Tech Stack  

- **HTML**  
- **Tailwind CSS**  
- **DaisyUI**
